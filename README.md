# BH
![image showing render made using this program](./image1.png)

## Introduction
Raytracer that generates png images.

I made this program because i wanted to make 3d simulations in C and i wanted something to display them.
Most of the time people use stuff like OpenGl of Vulkan to get the job done but i did not want to learn
all of that to make simple things. I like Raylib because of its simplicity in terms of it's usage but i
wanted something even more simple that i can say i made myself so that most of the project is of my own
making but does not require me having to learn OpenGl all over agian and failing at it and quiting shortly
after.

## Compiling
You can compile this program easily by just running the make command. you have to supply all the render
objects and add them to the scene in the main.c file and it will be rendered to a png. I will program it
to work as more so a utility more so in the future rather than a binary that just regenerates the same 
project again and again unless you compile it.
